## 4. AI モデル

### 4.1 使用モデル一覧

| モデル | Hugging Face ID | 種類 | 用途 |
|--------|----------------|------|------|
| 日本語モデル1 | christian-phu/bert-finetuned-japanese-sentiment | BERT（3クラス） | 日本語コメントの基本感情分析 |
| 日本語モデル2 | kit-nlp/bert-base-japanese-sentiment-irony | BERT（2クラス+皮肉検出） | 皮肉・反語の検出 |
| 多言語モデル | cardiffnlp/twitter-xlm-roberta-base-sentiment | XLM-RoBERTa（3クラス） | 日本語以外の感情分析 |

### 4.2 推論環境

| 項目 | 設定 |
|------|------|
| デバイス | CPU専用（GPU不要） |
| スレッド数 | 1（`torch.set_num_threads(1)`） |
| 最大トークン長 | 128 |
| バッチサイズ | 1（順次処理） |
| 勾配計算 | 無効化（`torch.no_grad()`） |
