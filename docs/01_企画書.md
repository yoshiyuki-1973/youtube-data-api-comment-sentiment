# 企画書

YouTube動画コメント感情分析システム

## 1. プロジェクト概要

YouTube動画のコメントを自動収集し、AI（3モデルBERTアンサンブル）とルールベース（200以上のパターン）を組み合わせた感情分析を行い、コメントのポジティブ・ネガティブ傾向を可視化するWebアプリケーションである。

## 2. 背景と目的

### 2.1 背景

YouTubeにおいて、動画のコメント欄はクリエイターと視聴者の重要なコミュニケーションチャネルである。しかし、コメント数が多い動画では全コメントを手作業で確認・分析することが困難であり、コメント全体の感情傾向を定量的に把握する手段が求められている。

### 2.2 目的

- YouTube動画のコメントに対する感情分析を自動化する
- 日本語コメントに特化した高精度な感情分析を実現する
- 分析結果をわかりやすいダッシュボードで可視化する
- ポートフォリオ作品として、AI/ML・API連携・Dockerコンテナ技術を実践的に組み合わせたシステムを構築する

## 3. ターゲットユーザー

| ユーザー | 用途 |
|---------|------|
| YouTubeクリエイター | 自身の動画に対する視聴者の反応を定量的に把握 |
| マーケティング担当者 | 特定の動画やキャンペーンに対する視聴者感情の分析 |
| データアナリスト | YouTube上のトレンドや感情傾向の調査 |
| ポートフォリオ閲覧者 | 技術力・設計力の評価 |

## 4. 主要機能

### 4.1 コメント収集

- YouTube Data API v3を使用した動画メタデータ・コメントの自動取得
- 複数のYouTube URL形式に対応（通常URL、短縮URL、Shorts、埋め込み）
- いいね数順でのコメントソート

### 4.2 感情分析

- 3モデルBERTアンサンブルによるAI感情分析
  - 日本語モデル2つ + 多言語モデル1つ
- 200以上のルールベースパターンによる補正
  - 皮肉・反語・否定表現・YouTube固有表現の検出
- 3値分類（ポジティブ / ネガティブ / ニュートラル）

### 4.3 可視化ダッシュボード

- Streamlit WebUIによるインタラクティブな結果表示
- 動画メタデータ（再生回数、高評価数等）のメトリクスカード表示
- 感情分析スコアの棒グラフ表示
- 感情フィルタ付きコメント一覧

### 4.4 CLIバッチ処理

- コマンドラインからの単一・複数動画一括分析
- バッチ処理結果のログ出力

## 5. 技術的特徴

| 項目 | 技術 |
|------|------|
| AI/ML | 3-model BERTアンサンブル + ルールベース(200+パターン) |
| 言語対応 | 日本語（2モデルアンサンブル）+ 多言語（XLM-RoBERTa） |
| WebUI | Streamlit |
| API連携 | YouTube Data API v3 |
| コンテナ | Docker / Docker Compose |
| クラウド | Azure Container Apps + Terraform（オプション） |
| テスト | pytest + カバレッジ計測 |
| 言語 | Python 3.11 |

## 6. システム構成概要

```
ユーザー
  │
  ├── Streamlit WebUI (http://localhost:8501)
  │     └── streamlit_app.py
  │
  └── CLI (docker compose exec)
        └── main.py
              │
              ▼
        YouTube Data API v3 → コメント取得
              │
              ▼
        3-model BERTアンサンブル → 感情分析
        + ルールベース(200+パターン)
              │
              ▼
        集計・可視化 → 画面表示（保存なし）
```

## 8. 制約事項

- YouTube Data API v3のクォータ制限（1日10,000ユニット）
- CPU専用推論（GPU不要、メモリ3.5GB以下）
- データの永続化は行わない（毎回APIから取得→分析→表示）
- 手動実行方式（スケジューラ不使用）
